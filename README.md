# PsygeneAnalyses

## To view and collaborate in github via Google Colab
Open the file using the "open in colab" button.
For importing to Github From Google Colab:
1. Link google colab to Github by navigating to settings, go to github and follow directions to link accounts.
2. Clear any outputs that contain widgets which will prevent files from correctly exporting to Github.
3. At the top of the page there will be blue text "Save in Github to keep changes", click it to bring up a pop up.
4. There will be two drop down fields to select from a repository and branch:
5.  Pushing to Repository: If you are trying to push to "KravitzLab/PsygeneAnalyses" you will need to be made a collaborater. Otherwise you can save to local or other repos under your own discretion.
6.  Branch: DO NOT SAVE TO MAIN. you need to create a new brach in "KravitzLab/PsygeneAnalyses" to save to.
